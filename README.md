# Daily Focus Timer & Task Log

A simple productivity web app for students to focus on one task at a time,so that they can compeletly focus on the work they are doing

## Features
- Focus timer (25/30/custom)
- Task-based focus sessions
- Daily focus log
- Clean and minimal UI

## Tech Stack
- HTML
- CSS
- JavaScript
